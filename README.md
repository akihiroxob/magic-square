# 魔法陣チャレンジ

iPad と Apple Pencil で遊べる 3×3 の魔法陣パズルです。手書き入力した数字を TensorFlow.js の手書き数字認識モデルで判別し、正しい魔法陣になっているか判定できます。GitHub Pages にそのまま公開できる静的サイトとして構成しています。

## 特徴

- Apple Pencil で書いた数字を自動認識（TensorFlow.js の MNIST モデルを利用）
- 手書きが難しいときのための手動入力欄付き
- 3×3 の魔法陣が完成しているかワンタップでチェック
- iPad や PC のブラウザからそのまま利用可能

## セットアップ

追加のビルド工程は不要です。リポジトリをクローンしたら、`index.html` をブラウザで開くだけで動作します。GitHub Pages を利用する場合は、このリポジトリをそのまま公開用ブランチに配置してください。

## ファイル構成

- `index.html` – 画面構造と UI
- `styles.css` – レイアウトとスタイル
- `app.js` – 手書き入力の処理、数字認識、魔法陣の判定ロジック
